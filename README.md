# Skyrim INI Helper

## Что это?
Skyrim INI Helper - небольшая программа для удобного управления .ini конфигарацией, с подробным описанием большинства параметров. 

## Поддерживаемые версии
Целевая версия - Skyrim Anniversary Edition, с возможными Role Play модификациями.

## Языки
Основной язык русский. Вы можете воспользоваться средствами машинного перевода для самостоятельного первода [описания параметров](https://github.com/whyimply/Skyrim-INI-Helper/blob/master/SIH/Resources/Data/ini_values_info.json).
